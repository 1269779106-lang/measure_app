# 手机测量器 - 操作手册

## 快速开始

### 安装

1. 手机连接电脑，开启 USB 调试
2. 运行：
   ```bash
   cd D:\mimo_soft\measure_app
   flutter run
   ```
3. 或安装 APK：`flutter build apk --release` 后安装 `build/app/outputs/flutter-apk/app-release.apk`

### 主页

打开 App 后看到 4 个功能卡片：

| 图标 | 功能 | 说明 |
|------|------|------|
| 📏 | 屏幕尺子 | 小物件精确测量 |
| 📐 | AR 测量 | 相机实时测距 |
| 🧭 | 水平仪 | 检测水平/垂直 |
| 📋 | 测量历史 | 查看历史记录 |

## 屏幕尺子

**适用场景**：测量小物件（硬币、钥匙、手指、USB 等）

**操作步骤**：

1. 点击主页 [屏幕尺子] 进入
2. 将被测物体放在手机屏幕上
3. **手指按住起点**，**拖动到终点**
4. 屏幕显示红色标记线
5. 松手后显示测量结果

**切换单位**：
- 点击右上角 [mm/cm/inch] 切换
- 支持毫米、厘米、英寸

**精度**：±0.5mm（取决于屏幕 PPI）

**技巧**：
- 物体边缘对准刻度线
- 多次测量取平均值
- 竖屏使用效果最佳

---

## AR 测量

**适用场景**：测量中等距离（桌子宽度、房间长度、家具尺寸）

**操作步骤**：

1. 点击主页 [AR 测量] 进入
2. **点击屏幕**放置起点（红点）
3. **再次点击**放置终点（红点）
4. 自动计算并显示距离

**重新测量**：
- 点击右上角 [刷新] 图标

**保存结果**：
- 点击 [保存] 按钮

**精度**：±2cm（需要良好光线和平坦表面）

**技巧**：
- 保持手机稳定
- 选择有纹理的表面（纯色墙面效果差）
- 测量距离建议 0.5m - 5m

---

## 水平仪

**适用场景**：检测桌面是否水平、墙面是否垂直、挂画对齐

**操作步骤**：

1. 点击主页 [水平仪] 进入
2. 将手机平放在待测表面上
3. 观察气泡位置：
   - **气泡居中** → 水平 ✅
   - **气泡偏移** → 倾斜 ⚠️

**读数**：
- 中央大数字：总倾斜角度
- 底部 X/Y：两个方向的分量

**校准**：
- 点击右上角 [校准] 将当前位置设为零点
- 在已知水平面上校准效果最佳

**精度**：±0.5°

---

## 测量历史

**功能**：
- 查看所有测量记录
- 显示测量类型、结果、时间
- 支持删除单条记录
- 支持清空全部记录

**操作**：
- 点击记录右侧 [删除] 图标删除单条
- 点击右上角 [清空] 图标清空全部

## 常见问题

**Q：测量结果不准怎么办？**
A：屏幕尺子的精度取决于屏幕 PPI，目前默认 160。可在设置中调整。AR 测量受光线和距离影响。

**Q：AR 测量点放不准？**
A：选择有纹理的表面，避免纯色/反光面。保持手机稳定。

**Q：水平仪不准？**
A：先在已知水平面上点击 [校准]。

**Q：如何导出测量数据？**
A：目前暂不支持导出，后续版本会添加。
